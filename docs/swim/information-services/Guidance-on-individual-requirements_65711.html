Guidance on individual requirements

Last updated:  JUNE 13th, 2019

This section of the handbook provides guidance on the requirements of the SWIM Service Description specification.

For each requirement, the supporting material is organised as:

  • Requirement. An exact copy of the requirement's table from the specification. 
    With an additional row "Level of Implementation" with values as indicated in "ANNEX B – Conformity Checklist";

  • Guidance. Such as textual explanation, example(s), etc.
