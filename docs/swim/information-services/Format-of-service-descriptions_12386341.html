Format of service descriptions

Last updated:  JUNE 28th, 2019

The SWIM Service Description specification prescribes the minimum set of elements to be contained in a service description. It doesn't prescribe any format for the service description. Whatever the format, it is important to be able to fulfill the requirements.

This section of the handbook provides a series of templates and schema that you can use to structure the content of service descriptions in various formats.

The Example service descriptions provides some example formats. It includes:

  • web page
  • Word document
  • XML file

The examples were based on the templates and schema available here.
