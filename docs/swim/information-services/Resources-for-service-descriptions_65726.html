Content of this section

  • Digital requirements — The SWIM Service Description specification in various digital means, in support of internal processes, such as service provider requirements management.
    • Requirements table (multi-purpose) — Table of the specification requirements, extending the Conformity table as in Annex B of the specification, to be used for various purposes within or outside the supporting material.
  • Conformance assessment — Assessing conformance to the specification.
    • Conformance assessment report — Conformance assessment report is used to capture the result of Conformance assessment.
  • Example service descriptions
    • Donlon TOBT Setting example
      • Donlon TOBT Setting Service Description
        • Conformance assessment - Donlon example — Conformance assessment for the Donlon TOBT Setting Service Description.
  • Format of service descriptions — This section of the handbook provides a series of templates and schema that you can use to structure the content of service descriptions in various formats.
